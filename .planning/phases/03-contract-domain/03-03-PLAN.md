    **Service — obligations.service.ts:**
    - Constructor injects PrismaService
    - `generateSchedule(contractId)`:
      1. Load contract with contractServices (include serviceDefinition), contractAreas, tenant
      2. If no contractServices, return early (no obligations to generate)
      3. Fetch active BillingPolicy for this airport: `prisma.billingPolicy.findFirst({ where: { airportId, isActive: true } })`. If none found, use default dueDateDays=30 and log warning.
      4. Generate monthly periods using pure date arithmetic:
         ```
         function generateMonthlyPeriods(effectiveFrom, effectiveTo):
           periods = []
           current = new Date(effectiveFrom.getFullYear(), effectiveFrom.getMonth(), 1)
           while current < effectiveTo:
             periodEnd = new Date(current.getFullYear(), current.getMonth() + 1, 0) // last day of month
             periods.push({ periodStart: new Date(current), periodEnd })
             current = new Date(current.getFullYear(), current.getMonth() + 1, 1) // 1st of next month
           return periods
         ```
      5. For each contractService, for each period:
         - Map serviceType to obligationType: rent->rent, revenue_share->revenue_share, service_charge->rent, utility->rent
         - Map serviceType to chargeType: rent->base_rent, revenue_share->revenue_share, service_charge->service_charge, utility->utility
         - Determine currency: contractService.overrideCurrency ?? contract.magCurrency ?? 'TRY'
         - Compute dueDate: new Date(periodEnd.getTime() + dueDateDays * 86400000) (or use date arithmetic)
         - Build obligation data object
      6. Use `prisma.obligation.createMany({ data: obligations })` for bulk insert
      7. Return count of created obligations

    - `findAll(query)`: paginated list with filters (contractId, tenantId, status, periodStart range). Include contract and tenant. Return { data, meta } shape.
    - `findOne(id)`: single obligation with contract, tenant, serviceDefinition relations. Throw NotFoundException if missing.

    **Listener — obligations.listener.ts:**
    ```typescript
    @Injectable()
    export class ObligationsListener {
      constructor(private readonly obligationsService: ObligationsService) {}

      @OnEvent('contract.published', { async: true })
      async handleContractPublished(payload: ContractPublishedEvent): Promise<void> {
        await this.obligationsService.generateSchedule(payload.contractId);
      }
    }
    ```
