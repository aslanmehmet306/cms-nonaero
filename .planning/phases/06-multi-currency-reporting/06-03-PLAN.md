    2. Create ReportsService:
       Inject PrismaService, ExchangeRatesService.

       **getDashboard(query: DashboardQueryDto)**:
       Run 5 parallel queries via Promise.all:
       a) Total revenue: prisma.obligation.groupBy({ by: ['currency'], where: { airportId, status: { in: ['ready', 'invoiced', 'settled'] }, ...periodFilters }, _sum: { amount: true }, _count: { id: true } })
       b) Outstanding invoices: prisma.invoiceLog.groupBy({ by: ['currency'], where: { airportId, status: { in: ['finalized', 'sent', 'past_due'] } }, _sum: { amountTotal: true }, _count: { _all: true } })
       c) Paid invoices: prisma.invoiceLog.groupBy({ by: ['currency'], where: { airportId, status: 'paid' }, _sum: { amountTotal: true }, _count: { _all: true } })
       d) Active contracts: prisma.contract.count({ where: { airportId, status: 'active' } })
       e) Active tenants: prisma.tenant.count({ where: { airportId, status: 'active' } })

       For collectionRate: sum all paid amounts (across currencies -- this is a rough KPI metric), sum all outstanding. collectionRate = paid / (paid + outstanding) * 100. If both zero, return 0.
       CRITICAL: Use `new Decimal(result._sum.amount?.toString() ?? '0')` for null-safe aggregation.

       Return DashboardResponse shape per research doc.

       **getRevenueSummary(query: RevenueSummaryQueryDto)**:
       a) By tenant: prisma.obligation.groupBy({ by: ['tenantId', 'currency'], where: { airportId, periodStart: { gte }, periodEnd: { lte }, status: { in: ['ready', 'invoiced', 'settled'] } }, _sum: { amount: true }, _count: { id: true } })
       Then join with tenant names: prisma.tenant.findMany({ where: { id: { in: tenantIds } }, select: { id: true, name: true } })

       b) By service: prisma.obligation.groupBy({ by: ['chargeType', 'currency'], where: { same filters }, _sum: { amount: true }, _count: { id: true } })

       If reportingCurrency is specified, convert each amount using exchangeRatesService.convert(amount, currency, reportingCurrency, periodTo date). Catch NotFoundException from convert (no rate found) and include unconverted amount with a warning.

       **getAgingReport(query: AgingReportQueryDto)**:
       Use $queryRaw with parameterized date (not CURRENT_DATE -- pitfall #5):
       ```sql
       SELECT
         CASE
           WHEN $1::date - due_date <= 0 THEN 'current'
           WHEN $1::date - due_date BETWEEN 1 AND 30 THEN '1-30'
           WHEN $1::date - due_date BETWEEN 31 AND 60 THEN '31-60'
           WHEN $1::date - due_date BETWEEN 61 AND 90 THEN '61-90'
           WHEN $1::date - due_date > 90 THEN '90+'
         END AS bucket,
         COUNT(*)::int AS count,
         SUM(amount_total) AS total_amount,
         currency
       FROM invoice_log
       WHERE airport_id = $2
         AND status NOT IN ('paid', 'voided')
         AND due_date IS NOT NULL
       GROUP BY bucket, currency
       ORDER BY bucket
       ```
       Note: Use `COUNT(*)::int` to avoid BigInt issue (pitfall #4).
       Convert BigInt counts via Number() as safety net.
       Convert total_amount via `new Decimal(row.total_amount?.toString() ?? '0')`.

       Return AgingReportResponse shape with summary totals.
