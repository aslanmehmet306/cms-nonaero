If EXCLUDE_CREATOR_KEY is true, fetch resource createdBy from request body or params, reject if matches user.sub (separation of duties). Return false with ForbiddenException for role failures.

       - **register.dto.ts**: email, password, name, role (IsEnum UserRole). Swagger decorators.
       - **refresh-token.dto.ts**: refreshToken (IsNotEmpty).

    4. **JWT Strategies**:
       - **jwt.strategy.ts**: PassportStrategy(Strategy, 'jwt'). Extract from Bearer token. Validate: lookup user by payload.sub via PrismaService, verify isActive. Return { sub, email, role, airportId, tenantId }. Set CLS context: userId, tenantId, airportId.
       - **jwt-refresh.strategy.ts**: PassportStrategy(Strategy, 'jwt-refresh'). Use JWT_REFRESH_SECRET (same as JWT_SECRET with "-refresh" suffix or separate env var — use JWT_SECRET for simplicity in dev). Extract from body field refreshToken. Validate: verify user exists and is active.

    5. **auth.service.ts**:
       - **login(email, password)**: Find user by email, verify password with bcrypt.compare, generate access + refresh tokens. Store refresh token hash in a simple approach: update user record with a hashedRefreshToken field (add to User model if needed) OR use Redis. For simplicity, store hashed refresh token on the User model — but since we should not modify the Prisma schema in this plan, store refresh tokens in Redis using ioredis. Key pattern: `refresh:{userId}` with TTL matching refresh expiration.
       - **refreshTokens(userId, refreshToken)**: Verify stored refresh token hash matches, generate new pair (rotation), store new hash, return new tokens. This is refresh token rotation.
       - **logout(userId)**: Delete refresh token from Redis.
       - **generateTokens(user)**: Create access token (15min, payload: sub, email, role, airportId, tenantId) and refresh token (7d, payload: sub).
       - Use bcrypt work factor 10 for password hashing.
       - Use @nestjs/jwt JwtService for token generation.

       - POST /api/v1/auth/logout — requires JWT, invalidate refresh token.
